GliderScore Direct Sync

Windows app that writes scored results directly into GliderScore's database — no CSV import needed. Run it on the CD's PC while connected to this base station.

  1. Download F3KSync.exe and place it anywhere on the CD's PC
  2. Double-click to open — browse to the GliderScore folder, enter this base station's URL
  3. Click Connect, choose the competition, then Sync Scores
  4. Click Recalculate in GliderScore to finalise normalised scores
Download F3KSync.exe
